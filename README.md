#  Rule-Based AI Chatbot

##  Project Overview

This project implements a simple Artificial Intelligence chatbot using rule-based logic.
The chatbot interacts with users by identifying predefined inputs and generating appropriate responses using conditional statements.

The main objective is to understand basic AI concepts like decision-making, control flow, and human-computer interaction.

## Features

- Handles greetings and user conversations
- Responds using predefined rules
- Supports exit commands
- Continuous interaction loop
- Simple and beginner-friendly AI implementation

##  Technologies Used

- Python
- Conditional Statements (if-else)
- Basic AI Logic

1. Clone the repository
